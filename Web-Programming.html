Chapter 6 - Web Technology - I

1. Introduction to the Internet

Definition: A global network of interconnected computers.

Components Required:

  • TCP/IP-enabled computer with a web browser.
  • ISP account.
  • Telephone line.
  • Modem.

Computer specifications:

  • 386 microprocessor, 16 MB RAM.
  • 256-color monitor (640x480 resolution).
  • 200 MB free hard disk space.
  • Multimedia sound card and speakers.

2. Internet Services & Applications

World Wide Web (WWW): Uses HTTP to link hypertext documents. Created in 1990 at CERN, Switzerland.

Web Browser:

Software to access and display web pages (e.g., Firefox, Chrome, Safari).

  • FTP: Transfers files between computers.
  • Email: Electronic messaging with multimedia support.
  • Voicemail: Records audio messages via phone.
  • USENET: Bulletin Board System with newsgroups.
  • Chat/IRC: Real-time text communication.
  • Video Conferencing: Real-time audio-visual communication.
  • E-Fax: Sends faxes via the internet (differs from email in cost and delivery).
  • Telnet: Remote login to another computer.
  • Navigators: Tools like Gopher, Archie, and WWW for information retrieval.

3. Key Differences: Email vs. E-Fax

Email E-Fax
Cheaper. More expensive.
Stored on the recipient’s server. Directly printed on the recipient’s printer.
Sent via local server. Sent via internet server.

4. Internet Terminologies

  • ISP: Provides internet access (e.g., WorldLink, Everest Net).
  • Link: Hyperlink to navigate between web pages.
  • Search Engine: Retrieves web content (e.g., Google, Yahoo).
  • Domain Name & IP Address:
    • Domain: Human-readable site identifier (e.g., example.com).
    • IP: Numeric address (e.g., 192.168.1.1).
  • DNS Server: Translates domain names to IP addresses.
  • Web Server: Hosts websites and delivers web pages.
  • POP/SMTP Servers:
    • POP: Retrieves emails from a server.
    • SMTP: Sends emails between servers.
  • Proxy Server: Enhances access speed and updates content.
  • Internet Roaming: Use local ISP account globally (e.g., user@wlink.com.np).

5. HTML Basics

Definition: HyperText Markup Language for creating web pages.

Versions: HTML 1.0 to HTML5.

Uses:

  • Design webpage layout.
  • Embed multimedia (images, audio).
  • Format text (bold, italic).
  • Create hyperlinks and lists.

Tags:

  • Pair Tags: Opening + closing (e.g., <p></p>).
  • Singular Tags: Self-closing (e.g., <br>).
  • Attributes: Modify tags (e.g., <body bgcolor="blue">).

Steps to Execute HTML:

  • Write code in Notepad.
  • Save as .html (e.g., sample.html).
  • Open via a web browser.

6. Web Design Terminology

  • Webpage: HTML document with text/images.
  • Website: Collection of linked webpages.
  • Homepage: Introductory page of a website.
  • URL: Address format (e.g., https://example.com).

7. DBMS (Database Management System)

Data vs. Information:

  • Data: Raw facts (e.g., "Ram, 19").
  • Information: Processed data (e.g., "Ram is 19 years old").

DBMS:

  • Database: Organized data collection (e.g., census records).
  • DBMS: Software to manage databases (e.g., MySQL, Oracle).

Advantages:

  • Reduces data redundancy.
  • Enhances security and sharing.
  • Quick retrieval.

Disadvantages:

  • High setup cost.
  • Requires technical expertise.
