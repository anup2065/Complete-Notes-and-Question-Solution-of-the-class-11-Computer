Chapter 4: Application Package

MS-Word (Word Processor)

MS-Word is a word processing software developed by Microsoft. It allows users to create, edit, and format documents efficiently. It supports features like spell check, thesaurus, mail merge, tables, and graphics.

Features of MS-Word

  • Fast operation: Digital text editing without mechanical movements.
  • Editing text: Supports insert, delete, modify, undo/redo, and formatting.
  • Storage: Documents can be saved electronically.
  • Graphics: Allows inserting images, graphs, and charts.
  • Mail merge: Automates the creation of multiple personalized documents.
  • Printing: Enables document printing anytime.
  • Spell check: Detects and corrects spelling and grammar errors.

MS-Excel (Spreadsheet)

MS-Excel is a spreadsheet program used for data storage, calculations, and analysis. It supports financial modeling, scientific calculations, and statistical analysis.

Features of MS-Excel

  • Supports multiple worksheets in a workbook.
  • Allows creating charts and graphs.
  • Facilitates sorting and organizing data.
  • Enables automatic calculations with formulas.
  • Supports object linking and embedding (OLE).
  • Includes formatting tools for improved presentation.

Cell Reference

Relative Reference: Adjusts when copied (e.g., A1, B3).

Absolute Reference: Fixed reference using "$" (e.g., $A$1, $B$2).

MS-PowerPoint (Presentation)

PowerPoint is a presentation software that helps create slides with text, images, charts, videos, and animations. It replaces traditional visual aids like posters and handouts.

Features of MS-PowerPoint

  • Supports themes, templates, and layouts.
  • Allows slide transitions and animations.
  • Enables the use of hyperlinks.
  • Includes tools for formatting text and graphics.
  • Supports online help and web linking.
