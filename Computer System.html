Chapter 1 - Computer System

1. Introduction to Computer System

Definition:

  • Electronic device derived from Latin "computare" (to calculate).
  • Processes raw data into meaningful information using IPO Cycle (Input → Process → Output).
  • Follows GIGO Principle (Garbage In, Garbage Out).

Block Diagram: Illustrates Von Neumann Architecture (Input Unit, CPU, Memory, Output Unit).

2. Characteristics of Computers

  • Accuracy: Error-free results (errors due to human input).
  • Speed: Operates in milliseconds/nanoseconds; processes millions of instructions per second.
  • Storage: Stores data in MB, GB, TB, PB (primary/secondary memory).
  • Versatility: Performs diverse tasks (education, business, engineering).
  • Diligence: Operates continuously without fatigue.
  • Automatic: Executes tasks without constant human intervention.

3. Applications of Computers

  • Education (e-learning, graphics).
  • Communication (email, video conferencing).
  • Business (invoicing, e-commerce).
  • Engineering (designing bridges, simulations).
  • Science, military, healthcare, robotics, transportation.

4. Capabilities and Limitations

Capabilities:

  • Handles large data volumes.
  • High-speed processing.
  • Solves complex problems (e.g., space exploration).

Limitations:

  • No decision-making ability.
  • Requires electricity/peripheral devices.
  • Health risks (eye strain).
  • Cannot learn or think like humans.

5. Generations of Computers

Generation Period Technology Features Examples
1st 1946–1958 Vacuum Tubes Large size, high heat, machine language. ENIAC, UNIVAC
2nd 1959–1964 Transistors Smaller, more reliable, assembly language. IBM 1401 (1st in Nepal)
3rd 1965–1974 Integrated Circuits (IC) Compact, commercial use, high-level language. IBM 360
4th 1975–1990 VLSI (Microprocessors) Portable, GUI, high memory. Apple Macintosh, IBM PC
5th 1990–onwards AI/ULSI Parallel processing, voice/image recognition. Modern AI systems

6. Components of Computer System

  • Hardware: Physical devices (keyboard, CPU).
  • Software: Programs (OS, MS-Office).
  • Data/Information: Raw input → Processed output.
  • People: Users/operators.
  • Procedure: Guidelines for system use.
  • Communication: Data flow between components.

7. Basic Hardware Components

  • Input Unit: Converts user input to binary (e.g., keyboard, scanner).
  • Output Unit: Converts binary to human-readable form (e.g., monitor, printer).
  • Hard Copy: Physical output (printers).
  • Soft Copy: Digital output (monitors).
  • Processing Unit (CPU):
    • ALU: Performs arithmetic/logical operations.
    • CU: Coordinates tasks.
  • Memory Unit: Stores data temporarily (RAM/ROM).
  • Storage Unit: Permanent storage (HDD, SSD, USB).

8. Memory Types

Primary Memory:

  • RAM: Volatile (SRAM: faster; DRAM: cheaper).
  • ROM: Non-volatile (PROM, EPROM, EEPROM).

Secondary Memory:

  • Magnetic: HDD, floppy disks.
  • Optical: CD, DVD, Blu-ray.
  • Flash: Pen drives, memory cards.
