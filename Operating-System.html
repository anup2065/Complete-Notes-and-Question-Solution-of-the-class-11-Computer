Computer Software and Operating Systems

1. Software

Definition: Collection of related programs and documents that instruct hardware to perform tasks.

Types:

System Software

Manages hardware and provides a platform for other software.

Categories:

  • System Management:
    • Operating System (OS): Controls overall computer operation (e.g., Windows, Linux).
    • Utility Software: Performs maintenance tasks (e.g., WinZip, Norton).
    • Device Drivers: Enable hardware-device communication (e.g., printer drivers).
  • System Development:
    • Programming Languages (e.g., Python, C).
    • Language Translators (compilers, interpreters, assemblers).

Application Software

Designed for specific user tasks.

Types:

  • Packaged Software: General-purpose (e.g., MS Office, Photoshop).
  • Customized Software: Tailored for specific needs (e.g., payroll systems).

2. Operating System (OS)

Definition: Integrated program managing hardware resources (CPU, memory, I/O devices).

Key Functions:

  • I/O Management
  • User Interface (UI)
  • Security (user authentication, virus protection)
  • Process Management
  • Memory Management
  • Data/File Management
  • Deadlock Prevention
  • Virtual Memory Management
  • Time-Sharing
  • Command Interpretation

3. Types of OS

A. Based on Processing:

  • Multi-programming: Concurrent execution of programs using a single CPU.
  • Multi-tasking: Multiple programs run simultaneously (e.g., Windows).
  • Multi-processing: Uses multiple CPUs (e.g., UNIX).
  • Time-Sharing: CPU time shared among users (quick response but security risks).
  • Batch Processing: Jobs processed in batches (e.g., payroll systems).
  • Real-Time OS: Time-critical tasks (e.g., flight control systems).
  • Online Processing: Immediate input/output (e.g., ATM transactions).

B. Based on User Interface:

C. Based on User Mode:

  • Single User: One user at a time (e.g., MS-DOS).
  • Multi User: Multiple users (e.g., Linux).

4. Examples of OS

Open Source: Linux, UNIX (modifiable source code).

Proprietary:

  • UNIX: Multi-user, multi-tasking (basis for Linux).
  • Linux: Open-source, supports multi-processing.
  • Mac OS: GUI-based for Apple devices.

5. Key OS Terminologies

  • Spooling: Manages I/O speed differences (e.g., print queues).
  • Process vs. Thread:
    • Process: Program in execution (states: running, ready, blocked).
    • Thread: Sub-process enabling concurrent execution.
  • Scheduling:
    • Priority: Jobs prioritized.
    • Time-Sharing: Equal CPU time allocation.
  • Buffering: Temporary storage during I/O operations.
  • Virtual Memory: Uses secondary storage to extend RAM.
